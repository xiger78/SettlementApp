## 1. 홈 화면 (메뉴)

앱을 실행하면 나타나는 첫 화면입니다.
- 상단에 **이번 달 정산 합계**와 모임 건수가 표시됩니다.
- 5개의 메뉴 카드: 모임정보등록 · 참가자등록 · 정산 · 월별정산일람 · 설정
- 아래에는 **최근 모임** 목록이 표시되며, 카드를 누르면 정산 화면으로, 「수정」을 누르면 모임정보를 편집합니다.

## 2. 모임정보등록

모임의 기본 정보를 등록합니다.
- **모임날짜**: 달력 아이콘을 눌러 날짜 선택
- **가게이름 / 가게전화번호** 입력
- **참가인원(총인원 · 남자 · 여자)**: 직접 입력하거나, 참가자등록 화면에서 이름을 추가하면 자동으로 갱신됩니다.
- **기타내용(메모)** 입력
- 하단의 **「등록하고 참가자 추가」** 버튼을 누르면 저장 후 참가자등록 화면으로 이동합니다.

<img src="screenshots/02_meetingform_ko.png" width="320" alt="모임정보등록" />

## 3. 참가자등록

선택한 모임의 참가자를 등록합니다. (최대 **30명**)
- 상단에 모임 정보(모임날짜 · 가게이름 · 등록 인원)가 표시됩니다.
- **이름**을 입력하고 **성별(남자/여자)**, **정산형태(현금/페이페이)**를 선택한 뒤 **「추가」** 버튼을 누릅니다.
- 등록된 참가자는 아래 **참가자 명단**에 표시되며, 성별·정산형태를 바로 바꾸거나 삭제할 수 있습니다.

<img src="screenshots/03_participants_ko.png" width="320" alt="참가자등록" />

## 4. 정산

영수증 금액을 입력하면 남녀 분담 금액이 자동으로 계산됩니다.

### 금액 계산 규칙
| 항목 | 계산식 |
| --- | --- |
| 정산금액 | 영수증 총액 (직접 입력) |
| 여자 합계 | 여자 1인 금액 × 여자 인원수 |
| 잔금액 | 정산금액 − 여자 합계 |
| 남자 1인 금액 | 잔금액 ÷ 남자 인원수 (자동 계산) |

> 예) 정산금액 40,000원, 여자 1인 5,000원, 남2·여2 → 여자 합계 10,000원, 잔금액 30,000원, **남자 1인 15,000원**

<img src="screenshots/04_settlement_calc_ko.png" width="320" alt="정산 금액 계산" />

### 영수증 촬영 · 참가자 정산
- **영수증**: 「촬영」 버튼으로 카메라를 실행하여 영수증 사진을 저장합니다.
- 참가자별로 금액과 정산형태가 표시되며, 정산이 끝난 사람은 왼쪽 원을 눌러 체크합니다.
- 마지막에 **「정산완료」** 버튼을 눌러 저장합니다.

<img src="screenshots/05_settlement_list_ko.png" width="320" alt="참가자 정산" />

## 5. 월별정산일람

월별 정산 합계를 한눈에 볼 수 있습니다.
- 상단에 **전체 정산 합계**가 표시됩니다.
- 월 카드를 누르면 해당 월의 모임 목록이 펼쳐지고, 모임을 누르면 정산 화면으로 이동합니다.

<img src="screenshots/07_monthly_ko.png" width="320" alt="월별정산일람" />

## 6. 설정 (언어 변경)

- **언어 설정**에서 **한국어** 또는 **日本語**를 선택합니다.
- 선택 즉시 메뉴와 모든 화면의 언어가 변경되며, 설정은 앱을 다시 켜도 유지됩니다.
